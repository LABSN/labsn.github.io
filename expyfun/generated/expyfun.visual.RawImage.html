expyfun.visual.RawImage¶

class expyfun.visual.RawImage(ec, image_buffer, pos=(0, 0), scale=1.0, units='norm')¶

Create image from array for on-screen display

Parameters:

ec : instance of ExperimentController

Parent EC.

image_buffer : array

N x M x 3 (or 4) array. Color values should range between 0 and 1.

pos : array-like

2-element array-like with X, Y (center) arguments.

scale : float

The scale factor. 1 is native size (pixel-to-pixel), 2 is twice as large, etc.

units : str

Units to use for the position. See check_units for options.
Returns:

img : instance of RawImage

The image object.
