expyfun.io.write_hdf5¶

expyfun.io.write_hdf5(fname, data, overwrite=False, compression=5)¶

Write python object to HDF5 format using h5py

Parameters:

fname : str

Filename to use.

data : object

Object to write. Can be of any of these types:

{ndarray, dict, list, tuple, int, float, str}

Note that dict objects must only have str keys.

overwrite : bool

If True, overwrite file (if it exists).

compression : int

An integer between 0 (no compression) to 9 (highest compression) using zlib.
