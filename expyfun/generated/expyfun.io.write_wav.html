    expyfun.io.write_wav¶

    expyfun.io.write_wav(fname, data, fs, dtype=<class 'numpy.int16'>, overwrite=False, verbose=None)[source]¶

    Write a WAV file

    Parameters:
    fnamestr

    Filename to save as.

    dataarray

    The data to save.

    fsint

    The sample rate of the data.

    dtypenumpy dtype

    The output format to use. np.int16 is standard for many wav files, but np.float32 or np.float64 has higher dynamic range.

    overwritebool

    If True, overwrite the file if necessary.

    verbosebool, str, int, or None

    If not None, override default verbose level.
