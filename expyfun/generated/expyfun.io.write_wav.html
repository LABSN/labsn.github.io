expyfun.io.write_wav¶

expyfun.io.write_wav(fname, data, fs, dtype=<type 'numpy.int16'>, overwrite=False, verbose=None)¶

Write a WAV file

Parameters:

fname : str

Filename to save as.

data : array

The data to save.

fs : int

The sample rate of the data.

format : numpy dtype

The output format to use. np.int16 is standard for many wav files, but np.float32 or np.float64 has higher dynamic range.
