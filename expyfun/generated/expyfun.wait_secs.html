expyfun.wait_secs¶

expyfun.wait_secs(secs, ec=None)[source]¶

Wait a specified number of seconds.

Parameters
secsfloat

Number of seconds to wait.

ecNone | expyfun.ExperimentController instance

Notes

This function uses a while loop. Although this slams the CPU, it will guarantee that events (keypresses, etc.) are processed.
