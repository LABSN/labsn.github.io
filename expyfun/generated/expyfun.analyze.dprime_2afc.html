expyfun.analyze.dprime_2afc¶

expyfun.analyze.dprime_2afc(hm, zero_correction=True)¶

Estimates d-prime for two-alternative forced-choice paradigms.

Parameters:

hm : array-like

Correct trials (hits) and incorrect trials (misses), in that order, as array-like data with last dimension having size 4.

zero_correction : bool

Whether to add a correction factor of 0.5 to each category to prevent division-by-zero leading to infinite d-prime values.
Returns:

dp : array-like

Array of dprimes with shape hmfc.shape[:-1].
