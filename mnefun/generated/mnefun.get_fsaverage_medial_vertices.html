mnefun.get_fsaverage_medial_vertices¶

mnefun.get_fsaverage_medial_vertices(concatenate=True)¶

Returns fsaverage medial wall vertex numbers

These refer to the standard fsaverage source space (with vertices from 0 to 2*10242-1).

Parameters:

concatenate : bool

If True, the medial wall vertices from the right hemisphere will be shifted by 10242 and concatenated to those of the left. Useful when treating the source space as a single entity.
Returns:

vertices : list of array, or array

The medial wall vertices.
